# 🌾 Farmers Market Web App

This is a Django-based web application where users can browse and manage products in a farmers market.

## 🚀 Features
- User Signup & Login
- Product Listing
- Order Management
- User Profile System

## 🛠️ Tech Stack
- Python
- Django
- HTML, CSS

## ▶️ Run Locally

```bash
git add .
git commit -m "deploy ready"
git push origin main

git clone https://github.com/chaudhary-priti06/farmers-market.git
cd farmers-market
pip install -r requirements.txt
python manage.py runserver
